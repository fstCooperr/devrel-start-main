DevRel API Workshop

This workshop helps beginners understand how APIs work using simple browser-based examples.

What you will learn

  • What an API is
  • How to make API requests from the browser
  • How to read JSON responses
  • Common beginner mistakes

How to start

  1. Open the project in VS Code
  2. Read the README.md file
  3. Follow the steps in the workshop folder

Everything is designed for beginners. No prior API experience is required.
